BOT WO - HƯỚNG DẪN
===================

1. Giải nén thư mục BOT_WO.
2. Đảm bảo Python đã được cài và đã chọn Add Python to PATH.
3. Nhấp đúp CAI_DAT.bat để cài thư viện (chỉ cần làm lần đầu).
4. Nhấp đúp CHAY_BOT.bat để chạy bot.
5. Trình duyệt sẽ mở giao diện BOT WO.

CẬP NHẬT DỮ LIỆU
- Thay file WoExport.xlsx trong thư mục BOT_WO bằng file mới, giữ nguyên tên.
- Hoặc tải file mới trực tiếp tại thanh bên của giao diện.
- Bấm "Làm mới dữ liệu" nếu vừa thay file.

QUY ƯỚC
- WO tồn = tất cả WO có trạng thái khác "Đóng".
- WO quá hạn = WO tồn có "Thời gian còn lại (H)" nhỏ hơn 0.
